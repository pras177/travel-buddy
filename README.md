# Travel Buddy 🌏

Multi-screen React Native app dengan React Navigation — destinasi wisata Indonesia.

## Screenshots
> Tambahkan screenshot setelah app berhasil jalan di Expo Go

## Features
- Bottom Tab Navigation — 3 tab: Home, Search, Favorites
- Stack Navigator di Home tab — HomeScreen → DetailScreen
- Route Params — data destinasi di-pass via `navigation.navigate()`
- 10 Destinasi Wisata Indonesia — image, harga, rating, deskripsi
- Search & Filter Kategori — real-time search + filter by kategori
- Add to Favorites — simpan/hapus destinasi favorit
- Favorites Badge — live count di tab icon
- @expo/vector-icons — Ionicons untuk tab icon dan UI
- Responsive Layout — Safe Area support iOS dan Android

## Tech Stack
- React Native + Expo (~54)
- React Navigation 6
- @react-navigation/native
- @react-navigation/bottom-tabs
- @react-navigation/native-stack
- StyleSheet
- @expo/vector-icons (Ionicons)
